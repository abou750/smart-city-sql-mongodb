# Smart City - SQL to MongoDB Migration

## Overview

Academic project completed as part of the Bachelor's Degree in Data Science (BUT Science des Données).

The objective of this project was to design a data management system capable of working with both relational and non-relational databases through an abstraction layer.

## Project Context

This project was carried out by a team of three students.

My contributions included:

* Data modeling
* SQL and MongoDB queries
* Data migration implementation
* Testing and validation

## Objectives

* Compare relational and NoSQL approaches
* Design a flexible data architecture
* Implement data persistence using SQL and MongoDB
* Create an abstraction layer between business logic and storage systems
* Facilitate migration between database technologies
